üöÄ Quick Start

This is an MCP (Model Context Protocol) service designed to work with AI assistants like Claude. To use it:

  1. Connect your AI assistant or MCP client to http://192.168.68.100:8011
  2. Use the available tools to search, create, and manage documentation
  3. External projects can connect and manage their own documentation
  4. All documentation is stored persistently and indexed for search

For more information, see the MCP Protocol Documentation
